Задание 1

Задача 1

Текст задачи

На основе списка вещественных чисел получить список целых чисел, отбросив

дробную часть.



Алгоритм решения

Программа начинает свое выполнение с функции main. Запрашивается у пользователя количество значений списка, которое сохраняется в переменную count. Затем введенное значение преобразуется в целое число. Выполняется проверка: если count меньше или равно 0, выводится сообщение «Ошибка: количество должно быть положительным!» и программа аварийно завершается с ошибкой через exit 1. Если count положительное число, вызывается функция readFloats для формирования списка вещественных чисел. Данная функция создает генератор списка, и запускается цикл for от 1 до count, где count это корректные данные. В каждой итерации цикла выводится сообщение «Введите вещественное число index: » с текущим номером элемента. Считывается строка, введенная пользователем, и преобразуется в вещественное число с помощью float. Полученное значение добавляется в формируемый список. После завершения цикла сформированный список вещественных чисел возвращается в функцию main и записывается в значение originalFloats. Затем с помощью функции List.map создается новый список integers, где каждый элемент исходного списка преобразуется в целое число с помощью int. В конце программы на экран выводятся оба списка: исходный список вещественных чисел и список целых чисел. Программа завершает работу с кодом возврата 0.

Задание 2

Задача 1

Текст задачи

Список содержит строки. Сколько из них имеет чётную длину?



Алгоритм решения

Программа начинает свое выполнение с функции main. Запрашивается у пользователя количество элементов списка, которое сохраняется в переменную count. Затем введенное значение преобразуется в целое число. Выполняется проверка: если count меньше или равно 0, выводится сообщение «Ошибка: количество должно быть положительным!» и программа завершается с ошибкой через exit 1. Если count положительное число, вызывается функция readStrings для формирования списка строк. Данная функция создает генератор списка, и запускается цикл for от 1 до count, где count это корректные данные. В каждой итерации цикла выводится сообщение «Введите элемент index: » с текущим номером элемента. Считывается строка, введенная пользователем, и полученное значение добавляется в формируемый список. После завершения цикла сформированный список строк возвращается в функцию main и сохраняется в переменную strings. Затем с помощью функции List.fold выполняется подсчет количества строк с четной длиной: для каждой строки проверяется, делится ли ее длина на 2 без остатка (s.Length % 2 = 0), и если условие выполняется, счетчик увеличивается на 1. Результат сохраняется в переменную evenCount. В конце программы на экран выводятся список всех введенных строк и количество строк с четной длиной. Программа завершает работу с кодом возврата 0.
